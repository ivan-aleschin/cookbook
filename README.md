# Книга рецептов

Моя личная кулинарная книга, сделанная в [Typst](https://typst.app/).  
Каждый рецепт — отдельный файл, книга собирается в красивый PDF одной командой.

## Скачать

Готовый PDF — в [Releases](/ivan-aleschin/cookbook/releases).

## Структура

Рецепты разбиты по разделам: супы, основные блюда, салаты, десерты.  
Каждый раздел оформлен в своей цветовой теме — томатный красный, авокадо, шоколадный и другие.

## Сборка

Требуется Typst ≥ 0.12.

```bash
# Разовая сборка
typst compile book.typ

# Live-режим (PDF обновляется при сохранении файла)
typst watch book.typ
```

Для просмотра PDF в реальном времени подойдёт [Zathura](https://pwmt.org/projects/zathura/) — он автоматически перезагружает файл при изменении.

## Добавить рецепт

1. Создать файл в нужном разделе, например `recipes/soups/my-soup.typ`
2. Скопировать структуру из любого существующего рецепта
3. Добавить `#include` в `book.typ`
